How do I get a token and repo?

1. Create a new public GitHub repo (e.g. "filedrop-storage").
2. Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token.
3. Give it the repo scope, then copy the token here. Treat it like a password.
